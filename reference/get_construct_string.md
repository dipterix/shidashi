# Get `R` expression used to generate the 'HTML' tags

This function only works on the elements generated by this package

## Usage

``` r
get_construct_string(x)
```

## Arguments

- x:

  'HTML' tags

## Value

Quoted `R` expressions that can generate the 'HTML' tags

## See also

[`format_text_r`](https://dipterix.org/shidashi/reference/format_text_r.md)

## Examples

``` r
x <- info_box("Message")
get_construct_string(x)
#> [1] "info_box(\"Message\")"
```
